# Known Limitations

## Model Limitations

- LLM accuracy depends on prompt quality and model version
- Some nuanced stances may be misclassified
- Ambiguous text requires human review

## Data Limitations

- Only processes English-language documents
- PDF extraction may fail on scanned documents
- Assumes consistent document structure

## Evaluation Limitations

- Metrics are approximations of true quality
- Golden outputs may contain errors
- Limited test coverage for edge cases
